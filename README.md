# Telegram Star Combiner

Темное веб‑приложение на **Eel + Telethon** для управления несколькими Telegram‑аккаунтами через единую панель.

## Возможности

- Добавление нескольких Telegram аккаунтов (по `api_id`, `api_hash`, `phone`).
- Запрос и подтверждение кода авторизации.
- Просмотр списка подключенных аккаунтов.
- Просмотр диалогов выбранного аккаунта.
- Отправка сообщений из выбранного аккаунта.
- Темный UI с градиентами, glow‑эффектами и анимациями.

## Быстрый старт

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Откроется окно Eel (или браузер) на `http://localhost:8000`.

## Важно

- Для работы нужны реальные `api_id` и `api_hash` с [my.telegram.org](https://my.telegram.org).
- Сессии Telethon сохраняются в папке `sessions/`.
- Это демонстрационный комбайнер: архитектура подготовлена для расширения (парсинг, рассылки, фильтры, массовые операции и т.д.).
