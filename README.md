# Telegram Star

Профессиональная и компактная панель Eel + Telethon для управления Telegram-аккаунтами.

## Ключевые изменения

- Исправлена ошибка event loop (`The asyncio event loop must not change...`) через единый фоновый loop.
- Аккаунты отображают имя профиля, а в описании формат: `@username | ID: 123` (или только `ID`).
- Подключение аккаунта происходит только при явных действиях пользователя (проверка, вход, отправка и т.д.).
- Импорт вынесен в удобное модальное окно с красивыми кастомными кнопками выбора файлов.
- Авто-импорт после выбора `.session`/`tdata`.
- Ручной вход отделен в отдельное окно "как регистрация", поле 2FA показывается только если реально нужно.
- Удаление аккаунтов из списка работает корректно.
- Добавлена защита от повторной загрузки одинаковой сессии (проверка по хэшу файла сессии).

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Папки

- `sessions/` — рабочие session-файлы
- `imports/` — загруженные tdata bundles
- `exports/` — экспорт
