# Telegram Star

Telegram Star — веб‑панель управления Telegram‑аккаунтами на **Eel + Telethon** в стиле Telegram Mini App (адаптация для ПК).

## Что реализовано

- Тёмный минималистичный интерфейс с мягкими градиентами, glass‑карточками и плавными hover/appear анимациями.
- Sidebar‑навигация с вкладками: **Главная / Аккаунты / Инструменты / Номера / Настройки**.
- Главная: hero‑блок, карточки статистики (валидные/спамблок/невалидные), сетка модулей.
- Аккаунты: поиск, расширенные фильтры (валидность, Premium, 2FA, username), табы статусов, карточки, пагинация.
- Модальное окно аккаунта: детальная информация, копирование username/ID, действия экспорта (tdata/telethon/pyrogram), теги.
- Настройки: профиль, создание/удаление кастомных тегов (localStorage), служебные пункты.
- Backend API (`app.py`) для управления аккаунтами Telethon: добавление, удаление, авторизация, диалоги, отправка сообщений.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

После запуска UI доступен через Eel на `http://localhost:8000`.

## Примечания

- Для реальной работы используйте собственные `api_id`/`api_hash` с [my.telegram.org](https://my.telegram.org).
- Telethon‑сессии сохраняются в `sessions/`.
